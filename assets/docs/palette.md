# Palette

This is the first-pass palette direction for `Rotten Borough: Municipal Gutworks`.

## Palette Roles

| Role | Hex | Use |
|---|---:|---|
| Foxman fur orange | `#c9682c` | Primary protagonist read. |
| Foxman dark fur | `#4b2419` | Ears, tail, contrast markings. |
| Ragged coat green-black | `#26352d` | Foxman clothing and silhouette support. |
| Tarnished brass | `#b88a3b` | Civic badges, buckles, UI trim. |
| Rotten parchment | `#d4b879` | UI panels, notices, readable labels later. |
| Sewer green | `#566b3b` | Background haze, bile-adjacent environment. |
| Toxic bile | `#a6d34a` | Hazards and poison VFX. |
| Bruised shadow | `#342a45` | Background depth and cool shadows. |
| Mud stone | `#5a5146` | Terrain base. |
| Hit red | `#b3312b` | Damage accents, not overused. |
| Bone highlight | `#e4d6a2` | Edge highlights and UI contrast. |
| Ink dark | `#161315` | Silhouettes and outline accents. |

## Readability Rules

- Foxman orange should not be used heavily in backgrounds.
- Toxic bile should be reserved for hazards, poison, and interactable danger.
- Hit red is for damage and urgent feedback, not general decoration.
- Backgrounds should favor mud stone, sewer green, bruised shadow, and tarnished brass.
- UI should use parchment, ink dark, brass, and controlled red/green accents.

## Lighting

Default lighting direction:

- Warm key from upper left.
- Cool sewer fill from lower right.
- Strong rim only for bosses, pickups, or interactable highlights.

## Contrast

- Player and enemies need the strongest silhouette contrast.
- Background parallax layers should be lower contrast and lower saturation.
- Foreground collision tiles need readable edges but should not fight characters.
